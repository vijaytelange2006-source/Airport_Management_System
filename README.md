# Airport Management System

## 📌 Description
This project is a DBMS-based application to manage airport operations like passengers, flights, and bookings.

## 🛠 Technologies Used
- Java
- MySQL
- JDBC

## ▶️ How to Run
1. Import project in VS Code
2. Setup MySQL database using airport.sql
3. Run Main.java
